# IM Spider

IM Spider는 **타일 설치와 로프 이동을 중심으로 한 퍼즐 액션 게임**입니다.  
플레이어는 게임을 클리어하기 위한 3개의 타일을 배치하고, 로프를 활용해 지형을 이동하며  
각 스테이지의 골인 지점에 도달해야 합니다.

퍼즐 설계와 이동 메커니즘의 결합을 핵심 재미 요소로 두고 있으며,  
단순한 조작 속에서 경로 선택과 판단을 요구하도록 구성했습니다.

---

## Key Features
- 타일 설치를 통한 지형 변화 퍼즐
- 로프 기반 이동 시스템
- 스테이지 단위의 퍼즐 액션 구조
- 물리 계산을 활용한 이동 및 충돌 처리

---

## Technology Stack
- **C++**
- **DirectX**
- Custom Game Engine

---

## Controls
- 이동: WASD
- 로프 / 액션: Mouse / Keyboard
